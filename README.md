# ReactJS Bangkok 1.0.0 - Universal Web Apps
A simple tutorial about building a universal web app using React and Redux

## Installation
```bash
npm install
```

## Running Dev Server
```bash
npm run dev
```

## Running Production Server

```bash
npm run build
npm start
```

## Deployment on Heroku

```bash
heroku login
heroku create
git push heroku master
heroku open
```
